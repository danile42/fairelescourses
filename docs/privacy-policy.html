Privacy Policy

App: Fairelescourses — Last updated: 2026-04-01

Fairelescourses is a supermarket navigation app. This policy explains what data the app stores, where it is stored, and how it is protected.

1. Summary

  • The app does not collect personal information such as your name, email address, or location.
  • Your shopping data (shops and lists) is stored on your device and never leaves it unless you explicitly enable household sync.
  • When sync is enabled, data is end-to-end encrypted before being sent to the cloud. The server cannot read your data.
  • No data is sold, shared with third parties, or used for advertising.

2. What is stored and where

Data Stored locally Stored in the cloud
Shops (layout, product placement) Always — even without sync Only when household sync is enabled
Shopping lists and items Always — even without sync Only when household sync is enabled
Household sync credentials Yes (your household code) Only a Firebase anonymous user ID
Personal information Not collected

3. Local-only mode

By default the app operates in local-only mode: all data stays on your device. Nothing is transmitted over the network. You can use the app fully without ever enabling sync.

4. Household sync (optional)

If you choose to join or create a household, your shops and lists are synchronised via Google Firebase Firestore. Before any data is sent to Firebase, it is encrypted on your device using your household code as the key. The server stores only ciphertext and cannot read your shopping data.

Firebase authentication uses an anonymous sign-in: no email address or password is required or stored. Firebase may collect anonymous usage metrics as described in Google's Firebase privacy policy.

5. OpenStreetMap / Overpass API

When you search for existing shops, the app queries the public Overpass API (OpenStreetMap data). Only the search query (a place name or category) is sent; no personal data is transmitted.

6. Permissions

The app requests no device permissions beyond internet access (required for optional sync and shop search).

7. Data retention and deletion

All local data is removed when you uninstall the app. If you used household sync, you can delete your cloud data at any time from the app's sync settings. Firebase anonymous accounts and their associated data are deleted upon request.

8. Children

The app does not knowingly collect data from children under 13.

9. Changes to this policy

Any updates to this policy will be reflected in the "Last updated" date above and, where material, noted in the Play Store release notes.

10. Contact

Questions about this policy can be raised via the app's GitHub issue tracker.
